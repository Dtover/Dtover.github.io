  2. npm installed
  3. nodejs installed

Step three

Some hexo commands

1. Generate static file

hexo g(enerate)

2. Start the local server

hexo s(erver)

3. Deploy your website

hexo d(eploy)

4. Create a new post

hexo new “Post_Name”
