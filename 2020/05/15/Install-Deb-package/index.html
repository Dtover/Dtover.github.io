Install Deb package

May 15, 2020

This file records the steps to install deb package in arch linux

1. Download Deb package
2. Extract with the following commands

ar x package.deb
xz -dk package.tar.zx
(-d: Extract)
(-k: Keep file after extracting)
pacman -U package.tar
